# Vc-Lab2

To Remove dev branch :-
<br/>
Locally:
git branch -d dev
<br/>
Remotely:
git push origin --delete dev


<br/><br/>

To List Tags :-
<br/>
git tag 

<br/><br/>

To Delete Tag :-
<br/>
Locally:
git tag -d v1.7
<br/>
Remotely:
git push --delete origin v1.7
